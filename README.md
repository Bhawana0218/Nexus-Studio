# Nexus Studio

A premium modern agency landing page built with Next.js, React, Tailwind CSS, React Bits, and 21st.dev inspired components.

## Overview

Nexus Studio is a frontend project created to practice modern UI/UX design principles using industry-standard tools and resources.

The project combines:

* Awwwards for design inspiration
* Figma for wireframing and UI design
* 21st.dev for modern component patterns
* React Bits for interactive animations
* Canva for branding assets and presentations

## Features

* Responsive Design
* Modern Hero Section
* Smooth Scroll Animations
* Interactive Components
* Project Showcase
* Testimonials Section
* Pricing Cards
* Contact Form
* Dark Mode Support

## Getting Started

Clone the repository:

git clone [https://github.com/your-username/nexus-studio.git](https://github.com/Bhawana0218/Nexus-Studio)

Install dependencies:

npm install

Run development server:

npm run dev

Open:

http://localhost:3000
